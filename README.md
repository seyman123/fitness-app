# Fitness App

Kullanıcıların kişisel fitness programlarını, günlük kalori ve BMI değerlerini takip edebileceği Flutter ile geliştirilmiş mobil uygulama.

## Özellikler (İlk Versiyon)
- Profil bilgileri girişi (yaş, cinsiyet, kilo, boy)
- BMI ve günlük kalori ihtiyacı hesaplama
- Dashboard ekranında özet gösterimi
- Light/Dark tema desteği (sistem temasına göre)

Bu çekirdek versiyon üzerine; antrenman programları, su ve beslenme takibi, istatistikler, bildirimler, sosyal paylaşım, Google Fit/Apple Health entegrasyonu ve çoklu dil için genişletme yapılacaktır.

## Çalıştırma

Proje kök dizininde:

```bash
flutter pub get
flutter run
```
